# 外交规则 v0.4

## §1 外交动作类型

每个 `diplomacy` 动作必须指定 `diplomacy_type`，6 种可选：

| 类型 | 消耗 | 效果 |
|------|------|------|
| `alliance_propose` | 0 | 向目标提议联盟（需信用 ≥ 50，不在背信冷却期） |
| `alliance_accept` | 0 | 接受对方发来的联盟提议，双方立即结盟 |
| `alliance_break` | 0 | 主动破盟（扣 30 信用，进入 5 tick 背信冷却） |
| `declare_war` | 0 | 宣战（被宣战方下一 tick 可看到你所有城精确兵力） |
| `trade_offer` | 0 | 贸易提议（v0.4 暂不实装具体结算，可作外交信号） |
| `message` | 0 | 纯文本喊话（无机制效果，仅公开可见） |

---

## §2 联盟系统

### §2.1 结盟流程

1. A 向 B 发送 `alliance_propose`
2. tick 结算后，B 的 state 中 `pending_alliance_from` = A
3. B 向 A 发送 `alliance_accept`
4. tick 结算后，双方建立联盟关系

### §2.2 联盟权利

- **信息共享**：联盟方互相看到对方所有城的精确兵力（不限于邻接城）
- **协同进攻**：双方在同一 tick 攻击同一目标时，攻击力合并计算
- **互不侵犯**：不能攻击盟友的城（违反自动破盟 + 扣 50 信用）

### §2.3 联盟约束

- 同时只能与一个势力结盟
- 结盟后 `your_alliance_with` 字段显示盟友
- `alliances` 全局列表公开可见（所有势力都能看到谁和谁联盟了）

### §2.4 破盟

- 任一方可通过 `alliance_break` 主动破盟
- 破盟方扣 **30 信用**
- 破盟方进入 **5 tick 背信冷却期**（期间所有 `alliance_propose` 被自动拒绝）
- 破盟事件记录在 `diplomacy_history`，所有势力可见

---

## §3 信用系统

### §3.1 信用值

- 初始：**100**
- 破盟：**-30**
- 盟期内攻击盟友：**-50**（且自动破盟）
- 恢复：连续 7 tick 未背叛后，每 tick **+5**（上限 100）
- 下限：0（不会负）

### §3.2 信用门槛

| 信用 | 效果 |
|------|------|
| < 50 | 所有 `alliance_propose` 被自动拒绝 |
| ≥ 50 | 可以提议联盟 |
| 100 | 满信用 |

### §3.3 背信冷却

- 破盟后 `betrayal_until = 当前 tick + 5`
- 冷却期内 `alliance_propose` 被自动拒绝
- 冷却期不影响其他外交动作（declare_war、message 等）

---

## §4 宣战系统

### §4.1 宣战效果

- 宣战方 A 对 B 宣战：B 的 state 中 `war_revealed_by = A`
- B 在下一 tick 可以看到 A **所有城池的精确兵力**（相当于边境侦察激活）
- `war_revealed_by` 仅持续 **1 tick**

### §4.2 未宣战的攻击

- 未宣战的势力攻击时，被攻击方**信息隔离仍然生效**
- 鼓励"明牌打"——宣战获取情报，但也有代价（对方也知道你要打了）

---

## §5 贸易系统 (v0.4 占位)

### §5.1 trade_offer

- `trade_offer` 类型已实现验证和存储
- 具体结算逻辑留给 Day 5+（粮草贸易、兵力调拨等）
- v0.4 中可用作外交信号（"我给你粮食"表达诚意，虽然暂不结算）

---

## §6 信息可见性

| 信息 | 自己 | 盟友 | 邻接方 | 宣战方 | 其他 |
|------|------|------|--------|--------|------|
| 自己城的精确兵力 | ✓ | ✓ | - | - | - |
| 盟友城的精确兵力 | - | ✓ | - | - | - |
| 邻接城的精确兵力 | - | - | ✓ | - | - |
| 宣战方城的精确兵力 | - | - | - | ✓(1tick) | - |
| 远处城的模糊兵力 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 联盟关系 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 信用分（自己的） | ✓ | - | - | - | - |
| 外交历史 | ✓ | ✓ | ✓ | ✓ | ✓ |
